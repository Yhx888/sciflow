科研流程编排器 — 用自然语言串联你的整个研究工作流

MCP/Skill 驱动 · 命令行优先 · 工具生态串联

💡 一句话介绍

一款基于 MCP 协议的科研工作流编排CLI工具，通过自然语言指令无缝串联文献管理（Zotero）、技术文档（Lark）、数据分析（Python）、论文撰写（Word）等工具，实现科研全流程自动化。

🔄 工作流架构

📚 文献调研 Trae Skill
→
📦 文献管理 Zotero MCP
→
📝 技术文档 Lark CLI
→
🐍 数据分析 Python MCP
→
📄 论文撰写 Word MCP

⚡ 核心功能

🔍

智能文献调研

调用 Trae Skill 进行网络检索，自动解析论文并提取关键信息，一键同步至 Zotero 文献库

📊

自动化文档生成

基于 Zotero 文献库自动生成技术调研文档、文献综述、研究背景章节

🔬

实验方案辅助

读取技术文档，结合研究目标生成实验方案建议，辅助研究设计

📈

数据处理管道

自动执行 Python 数据处理脚本，生成图表并格式化结果

✍️

论文文档导出

将处理结果、图表、结论自动填充至 Word 模板，生成可投稿文档

🔗

MCP 工具串联

通过 MCP 协议连接 Zotero、Lark、Python、Word 等工具，保证稳定性

👤 典型用户流程

  1. "帮我调研 ROS2 导航相关的最新论文"
    → Trae Skill 检索 arXiv/Google Scholar → 自动解析摘要 → 一键入库 Zotero
  2. "生成技术调研文档"
    → 读取 Zotero 文献 → Lark CLI 创建云文档 → 生成 Markdown 技术报告
  3. "分析现有方案的优缺点"
    → 读取 Lark 文档 → 结合文献 → 生成实验方案建议
  4. "处理实验数据，生成对比图表"
    → Python MCP 执行数据处理 → 自动生成 PNG/SVG 图表
  5. "导出论文初稿"
    → Word MCP 填充内容 → 生成格式规范的论文文档

🎯 技术栈

Python 3.10+ MCP Protocol Trae Skill Zotero API Lark CLI python-docx Click CLI Pydantic
